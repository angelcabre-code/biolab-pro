# Biolab Pro 🚀

Biolab Pro is a high-precision biomechanics analysis platform designed for sports performance and clinical assessment. This project is now **Open Source** to support the community of researchers, coaches, and developers.

## 🔗 Sinergia Teacher Integration

Biolab Pro features elective synchronization with **Sinergia Teacher**.
> [!IMPORTANT]
> Sinergia Teacher is a proprietary and protected professional system. While Biolab Pro is open source, the integration endpoints and data handling protocols with Sinergia are restricted to maintain the security and integrity of the professional ecosystem. This functionality is provided "as-is" for existing Sinergia users.

## 📱 Mobile & Desktop
Biolab Pro is cross-platform, supporting both **Windows (Electron)** and **Android (Capacitor)**.

## 🛠️ Tech Stack
- **Frontend**: Vue 3 + Vite + TypeScript
- **Styling**: Tailwind CSS
- **Desktop**: Electron
- **Mobile**: Capacitor
- **Backend/Sync**: Firebase + Sinergia Cloud Functions
- **Hardware**: ESP32 integration via WebSocket/HTTP
